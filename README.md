# im_h5_login_demo

一个 vue 工程，用于展示 open im sdk 的接入方式。

## 运行

open im 的 sdk 初始化时需要传入 open im server 的网址。若要自定义 open im server 的网址，请将 `.env` 文件复制成 `.env.local` 文件，并修改后端 ip。

1. `pnpm i`
2. `pnpm dev`
